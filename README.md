📈 SB Stocks – Stock Trading Dashboard

SB Stocks is a Full Stack Stock Trading Web Application built using the MERN Stack.
It allows users to track stocks, buy and sell stocks, and manage their portfolio through an interactive dashboard.

🔗 Live Demo:
https://samiksha-sb-stocks.netlify.app/dashboard

🚀 Features

User Authentication (Login / Register)

Stock Dashboard

Buy / Sell Stocks

Transaction History

Portfolio Management

Admin Dashboard

Real-time Stock Data API Integration

Responsive UI

🛠 Tech Stack
Frontend

React.js

Redux Toolkit

Axios

Vite

Tailwind / CSS

Backend

Node.js

Express.js

JWT Authentication

Database

MongoDB

Deployment

Frontend: Netlify

Backend: Render

Repository: GitHub
